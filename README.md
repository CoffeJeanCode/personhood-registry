# Registro de Usuarios Personería

Aplicación para hacer un historial de usuarios.

## Tecnologías

- Java
  - Maven
  - Data JPA
  - Thymeleaf
- MySQl
- Docker Compose (Dev)

## Modelos

#### Usuario (User)

##### Campos

- Nombres (FirstName)
- Apellidos (FirstName)
- Correo Electrónico (email)
- Teléfono (phoneNumber)
- Género (gender)
